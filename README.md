# 📊 CafeDashBoard

Um sistema simples feito com **Streamlit** e **Pandas** para gerenciar clientes, produtos e vendas, com relatórios visuais.

## 🚀 Funcionalidades
- Cadastro, edição e exclusão de clientes.
- Cadastro, edição e exclusão de produtos.
- Registro de vendas associadas a clientes e produtos.
- Relatório de consumo mensal com gráficos.
- Cálculo de faturamento no período selecionado.

## 🛠 Tecnologias
- [Python 3.x](https://www.python.org/)
- [Streamlit](https://streamlit.io/)
- [Pandas](https://pandas.pydata.org/)
